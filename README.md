# deathwatch-viewwer

viewer for the jsonl files generated by [deathwatch](https://github.com/canine-systems/deathwatch).

## Usage

the Caddyfile assumes the following setup:

- `/usr/share/deathwatch/` contains the contents from the `./deathwatch` folder generated by the mod
- `/usr/share/deathwatch-viewer/` contains the contents of this repository
